Last Updated: 11/05/2021

What is "Registration Form"?

Registration Form is a simulated website that allows users to input their personal data as they would when signing up for a website.

Features include:

  • Usable text input
  • Sign in and register buttons
  • Nicely formatted user interface
  • And much more!

Desired features include:

  • Storing info to actually create an account
  • Ability to view others who have created accounts
  • A Profile Page

What languages/platforms will be used?

I will be utilizing HTML, CSS, and JavaScript for this project.

Technical features learned in class:

  • HTML/CSS/JavaScript
  • Animations
  • Buttons
  • JavaScript Functions

    • Technical features beyond what I learned in class:

    • User input
    • Refined Interface
    • User input storage
